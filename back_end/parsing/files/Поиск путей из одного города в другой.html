На рисунке  — схема дорог, связывающих города А, Б, В, Г, Д, Е, Ж, К и Л. По каждой дороге можно двигаться только в одном направлении, указанном стрелкой. Сколько существует различных путей из города А в город Л?

Решение.

Количество путей до города X  =  количество путей добраться в любой из тех городов, из которых есть дорога в Х.

С помощью этого наблюдения посчитаем последовательно количество путей до каждого из городов:

А = 1;

Б = А = 1;

В = А + Б = 1 + 1 = 2;

Д = А = 1;

Г = А + Д = 1 + 1 = 2;

Е = Б + В = 1 + 2 = 3;

Ж = Д + Г = 1 + 2 = 3;

З = В + Г + Е + Ж = 2 + 2 + 3 + 3 = 10;

И = Е = 3;

К = Ж = 3;

Л = Е + Ж + З + И + К = 3 + 3 + 10 + 3 + 3 = 22.

 

Ответ: 22.
